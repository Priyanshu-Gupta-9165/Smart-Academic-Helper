## ✨ Features

### 🤖 BrainBuddy Bot — AI Assistant
- 24/7 AI-powered chatbot for academic questions
- Supports **Markdown rendering** with formatted responses
- **LaTeX/KaTeX** equation rendering for math and science
- **Code syntax highlighting** with Fira Code font
- Voice input support via microphone

### 📊 SGPA Calculator
- Calculate your Semester Grade Point Average easily
- Dynamic subject fields — add as many subjects as needed
- Clean, intuitive interface

### ✅ Todo List
- Track assignments and deadlines
- Filter tasks: **All**, **Active**, **Completed**
- Clear completed tasks in one click

### 💬 Feedback System
- Star rating system
- Text feedback with confetti celebration on submit

### 🎨 UI/UX
- **Glassmorphism** design with blur effects
- **Light/Dark mode** toggle
- Smooth animations and transitions
- Fully responsive layout
- Inspirational quotes scroller

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Priyanshu-Gupta-9165/Smart-Academic-Helper.git
   ```

2. **Open the project**
   ```bash
   cd Smart-Academic-Helper
   ```

3. **Launch**
   - Open `index.html` in your browser, or
   - Use a live server extension (e.g., VS Code Live Server)

## 📁 Project Structure

```
Smart-Academic-Helper/
├── index.html      # Main HTML page
├── styles.css      # All styles (glassmorphism, themes, responsive)
├── chatbot.js      # AI chatbot logic & API integration
├── script.js       # Theme toggle & general scripts
├── sgpa.js         # SGPA calculator logic
├── todo.js         # Todo list functionality
├── quotes.js       # Inspirational quotes data
└── README.md       # Project documentation
```

## 🛠️ Tech Stack

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, glassmorphism, animations
- **JavaScript (ES6+)** — Modular scripts
- **Font Awesome** — Icons
- **Google Fonts** — Poppins & Fira Code
- **KaTeX** — Math equation rendering
